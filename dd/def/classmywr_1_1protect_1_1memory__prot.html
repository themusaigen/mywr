enum mywr::protect::memory_prot::Enum : std::uint32_t

Enum that contains all possible memory`s protection constants for Windows and Linux.

Enumerator
kUnknown 

Unknown memory protection. It can be interpreted as an error.

kNoAccess 

There is no access to this memory section. It is not possible to execute/read/write.

kReadOnly 

A read-only memory section. Windows only.

kWriteCopy 

Copy-on-write access to a mapped view of a file mapping object. An attempt to write to a committed copy-on-write page results in a private copy of the page being made for the process. Windows only.

kExecute 

The memory section intended for execution.

kRead 

The memory section intended for reading.

kWrite 

The memory section intended for writing.

kReadWrite 

A set of memory flags that is intended for writing and reading.

kExecuteRead 

set of memory flags that is intended for executing and reading.

kExecuteReadWrite 

A set of memory flags that is intended for executing, reading and writing.

kExecuteWriteCopy 

A set of memory flags that is intended for executing, reading and copy-on-writing. Windows only.
