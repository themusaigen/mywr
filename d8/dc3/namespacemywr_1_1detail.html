Converts the calling convention according to the current system ABI.

Each architecture or system has its own unique calling convention. On Windows 32 bit, there are basic conventions: cdecl, stdcall, thiscall, fastcall, and so on. On Unix-like 32 bit, there is only one calling convention: cdecl. On Windows and Unix-like systems with 64-bit architecture, there is only one calling convention used by all functions.
